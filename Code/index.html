Step 1 — Serve this page

Browsers block localhost calls from a file:// page (CORS). Serve it:

python3 -m http.server 8080

Then open http://localhost:8080/ in your browser.

Step 2 — Start Ollama with CORS enabled

# macOS / Linux OLLAMA_ORIGINS="*" ollama serve
# Windows (PowerShell) $env:OLLAMA_ORIGINS="*"; ollama serve

Step 3 — Pull models

ollama pull deepseek-r1:7b ollama pull qwen2.5-coder:7b ollama pull gemma3:4b ollama pull qwen3:4b ollama pull llama3:8b ollama pull phi3:mini
